# Creative Ideas Generator 🧠✨

一个创意想法生成器，提供100个多样化的创意想法，支持PayPal付费支持创作者。

## 功能特性

- 🎯 **100个精选创意想法** - 涵盖12个不同类别
- 💡 **随机想法生成** - 获取即时创意灵感
- ❤️ **想法点赞系统** - 支持喜欢的想法
- 💰 **PayPal支付集成** - 支持创作者($1/想法)
- 📊 **实时分析** - 用户行为追踪
- 📱 **响应式设计** - 支持所有设备
- 🌐 **SEO优化** - 针对美国/欧盟市场

## 技术栈

- **前端**: HTML5, Tailwind CSS, Vanilla JavaScript
- **后端**: Node.js (本地开发)
- **部署**: Vercel (静态托管)
- **支付**: PayPal.me集成
- **分析**: 自定义事件追踪

## 本地开发

```bash
# 启动服务器
node server.js

# 访问地址
http://localhost:3001
```
